    快速启动

    1. 下载完整代码

      git clone https://github.com/XiaoMi/thain.git

    2. 运行环境需求

      • jdk >= 8
      • maven 3
      • nodejs >= 8
      • 操作系统：Linux 或 MacOS 测试通过，windows前端编译 可能 会有问题

    3. 在项目根目录下依次执行

      cd thain-fe
npm install
npm run build
cd ..
mvn -U clean package

    4. 打包完成后，即可运行

      java -jar thain-server/target/thain-server-1.1.5.jar

    5. 打开 localhost:9900 就可以看到效果了

    6. 初始账号密码：admin, admin
