    HTTP请求组件

    组件名称

    std::http

    必填参数

    url: url

    method: GET 或 POST

    选填参数

    contentType: Content-Type

    referenceData: 流程数据引用, 多个用逗号分开, 如: aName:a.name,bAge:b.age, 冒号前面的是http发过去的key, 后面是value

    forwardData: 转发数据, 这个会原封不动的发给url, key是forwardData

    产生结果

    result : http请求的返回结果
