    Shell脚本执行组件

    组件名称

    std::shell

    必填参数

    shellBase64: shell 脚本文件 base64

    选填参数

    environmentVariable: shell 变量（xxx=yyy 形式，多个用换行隔开）

    其他

    推荐在页面创建时候使用该组件, 如果是第三方系统建议用http在自己的系统实现
